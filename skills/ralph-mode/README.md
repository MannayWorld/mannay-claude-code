# Ralph Mode Skill

Autonomous loop execution for long-running development tasks using PRD-based user stories.

## Quick Start

1. Create PRD: `scripts/ralph/prd.json`
2. Initialize progress: `scripts/ralph/progress.txt`
3. Start Ralph: `/ralph-start`
4. Monitor: `/ralph-status`
5. Stop if needed: `/ralph-stop`

## See SKILL.md for Complete Documentation

The SKILL.md file contains comprehensive instructions that Claude uses during execution.
